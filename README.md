# Cleany
Chrome extension that clears your browser every time you open it. Perfect for public computers.

## Características / Features
- Limpieza automática de cookies, historial, sesiones y contraseñas.
- Automatic cleaning of cookies, history, sessions, and passwords.
- Selector de idioma (Español/English).
- Language selector (Spanish/English).
- Privacidad total al iniciar el navegador.
- Total privacy when starting the browser.
